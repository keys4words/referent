The application receives the address of an article in any language at the input, performs parsing and, using the appropriate button, connects the AI and performs the desired action. All results are generated in Russian.

Interface:

The URL input field for an article in any language.

3 buttons:  «О чем статья?» (blue), «Тезисы» (green), «Пост для Telegram» (red).

The block for displaying the result, in which the desired type of response is generated in Russian.
