The application receives the address of the English-language article at the input, performs parsing and, using the appropriate button, connects the AI and performs the desired action.

Interface:

The URL input field of the English-language article.

3 buttons:  «О чем статья?» (blue), «Тезисы» (green), «Пост для Telegram» (red).

The block for displaying the result, in which the desired type of response is generated.
